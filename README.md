Kitap Barkod Oluşturucu
Bu proje, kütüphane otomasyon sistemlerinden (örneğin Koha) dışa aktarılan .csv veya .xlsx dosyalarındaki verileri kullanarak standartlara uygun kitap sırtı etiketleri ve barkodlar oluşturmak için geliştirilmiş bir React uygulamasıdır.

Özellikler
Dosya Yükleme: .csv ve .xlsx formatındaki veri dosyalarını destekler.

Esnek Etiket Tasarımı: Etikete basılacak veri alanları (Yer Numarası, Başlık vb.) seçilebilir.

Özelleştirme: Yazı tipi boyutu, hizalama, logo ekleme gibi ayarlar mevcuttur.

Şablon Desteği: Farklı etiket kağıtları için hazır şablonlar ve özel şablon kaydetme özelliği.

Toplu Seçim: Veri tablosundan tek tek, sayfa sayfa veya barkod aralığına göre toplu seçim yapma.

PDF Olarak İndirme: Oluşturulan etiket sayfasını yazdırmaya hazır A4 formatında PDF olarak indirir.

Karanlık Mod: Daha konforlu bir kullanım için arayüzde karanlık mod desteği.

Kurulum ve Çalıştırma
Projeyi Klonlayın:

git clone [https://github.com/KULLANICI_ADINIZ/kitap-barkod-uygulamasi.git](https://github.com/KULLANICI_ADINIZ/kitap-barkod-uygulamasi.git)
cd kitap-barkod-uygulamasi

Bağımlılıkları Yükleyin:

npm install

Geliştirme Sunucusunu Başlatın:

npm run dev

Uygulama varsayılan olarak http://localhost:5173 adresinde çalışacaktır.

Yayınlama (Deployment)
Bu proje Vercel ile kolayca yayınlanabilir.

Projenizi GitHub'a yükleyin.

Vercel'e gidin, GitHub hesabınızla giriş yapın.

Proje deposunu Vercel'e ekleyin.

Vercel, projenin bir Vite projesi olduğunu otomatik olarak algılayacak ve gerekli ayarları yapacaktır.

"Deploy" butonuna basın ve işlemin tamamlanmasını bekleyin.
